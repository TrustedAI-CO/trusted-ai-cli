# エントリーガイドライン

## 自己推薦文について

**重要:** 発注者が見るのは自己推薦文のみ。この文章で自社の強みを伝える。

## トーン

- 丁寧かつ簡潔
- 具体的な数値・実績で説明
- 顧客の課題解決にフォーカス

## 自己推薦文テンプレート

### 基本構成（150〜300字目安）

```
[1文目: 案件への関心・適合性]
[2-3文目: 関連する強み・実績]
[最終文: 商談への意欲]
```

### AI案件向け

```
本案件に強い関心を持ちエントリーいたします。
弊社はAI開発専門企業として、チームの60%がAI PhDを保有し、
CVPR・NeurIPS等の世界トップカンファレンスでの採択実績がございます。
[案件に関連する事例: 例「社内ドキュメント検索AI」「異常検知AI」等]
の開発実績もあり、お客様の課題解決に貢献できると考えております。
ぜひ詳細をご説明させていただければ幸いです。
```

### システム開発案件向け

```
本案件に強い関心を持ちエントリーいたします。
弊社はAI×システム開発を一気通貫で対応しており、
東京大学・東京工業大学出身のエンジニアが在籍しております。
Amazon、LINEヤフー、野村総研での大規模システム開発経験を活かし、
設計から運用まで責任を持ってサポートいたします。
ぜひ詳細をご説明させていただければ幸いです。
```

### 製造業向け

```
本案件に強い関心を持ちエントリーいたします。
弊社は製造業向けAIシステムの開発実績があり、
半導体異常検知（検査速度1000倍向上）、不良品検出（精度99.995%）
などの成果を上げております。
お客様の品質管理・生産性向上に貢献できると考えております。
ぜひ詳細をご説明させていただければ幸いです。
```

## NGワード

- 「絶対に」「必ず」 — 保証できない
- 「簡単です」「すぐできます」 — 軽く見られる
- 「なんでもできます」 — 専門性が伝わらない
- 「業界No.1」 — 根拠なき主張

## 商談メッセージ

### 返信の基本構成

1. お礼
2. 質問への回答
3. 次のステップ提案

### 例

```
お問い合わせありがとうございます。

[質問への回答]

詳細をご説明させていただければと存じますので、
オンラインでの商談のお時間をいただけますでしょうか。

ご検討のほど、よろしくお願いいたします。
```
