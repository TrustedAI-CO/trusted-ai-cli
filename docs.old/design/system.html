System Design

Overview

The tai CLI is a Typer-based Python application with a layered architecture:

  • Entry point (main.py) — Typer app, global options, plugin loader
  • Commands (commands/) — one module per command group (auth, config, claude, pdf, hub, etc.)
  • Core library (core/) — shared services (auth, config, http, keystore, errors)
  • Data (data/) — bundled templates and brand assets
  • Plugins — entry-point discovery for third-party extensions
