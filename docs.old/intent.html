Product Intent

Context

TrustedAI CLI (tai) is a command-line tool for the TrustedAI team. It centralizes authentication, configuration, AI-assisted workflows, PDF generation, and Claude Code skill management into a single developer tool.

Problem

Team members juggle multiple tools, APIs, and manual processes. Authentication, document generation, secret management, and AI workflows each require separate setup and context switching.

Solution

tai provides a unified CLI with Google OAuth, profile-based config, secret management (keychain/file/env), Typst PDF generation, Claude Code skill installation, and a plugin architecture for extensibility.

Success Criteria

  • All team members use tai for daily workflows
  • Zero manual secret handling — everything goes through keystore
  • Claude Code skills are installable with one command
